📚 Data Structures in C
A comprehensive collection of Data Structures and Algorithm implementations written in C, organized by lab cycles.

🧠 Topics Covered
The programs across cycles cover fundamental data structures and algorithms, including:

Arrays – Searching, sorting, and traversal
Linked Lists – Singly, doubly, and circular linked lists
Stacks – Array and linked list based implementations
Queues – Linear, circular, and priority queues
Trees – Binary trees, BST, tree traversals
Graphs – BFS, DFS, adjacency matrix/list representations
Sorting Algorithms – Bubble, selection, insertion, merge, quick sort
Searching Algorithms – Linear search, binary search
Hashing – Hash tables and collision handling


🛠️ Prerequisites

GCC Compiler
Any Linux/macOS terminal or Windows with MinGW/WSL


▶️ How to Run

Clone the repository:

bash   git clone https://github.com/Gowry03/DataStructures.git
   cd DataStructures

Navigate to the desired cycle:

bash   cd cycle1

Compile and run a program:

bash   gcc filename.c -o output
   ./output
